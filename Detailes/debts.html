تسجيل دين كلي/جزئي

أضف الدين كاملًا أو جزءًا منه عند الاشتراك/التجديد—مع تحديث الرصيد تلقائيًا بعد كل دفعة.

سجل دفعات مفصل

تاريخ، مبلغ، عملة، ملاحظات، وإشعار واتساب لكل دفعة—كلها مرتبطة بالمشترك والدين.

Filter

فلترة دقيقة

استحقاق/أيام التأخير، الحالة (مدفوع/جزئي/متأخر)، والعملة—بحث سريع بلا عناء.

كيف يعمل النظام؟

  1. تسجيل الدين: كلي أو جزئي مع تحديد تاريخ الاستحقاق والعملة.
  2. دفعات مرتبطة: كل دفعة تخصم الرصيد وتُحفظ كسجل مستقل.
  3. المتابعة: لوحة تعرض المتأخر والجزئي والجديد مع مؤشرات يومية.
  4. الإشعارات: تأكيد دفع للعميل ومطالبات تلقائية للمتأخرين.
  5. التقارير: تصدير Excel محمي يشمل الفلاتر وتاريخ التصدير.
🧠
يُستثنى الدين تلقائيًا من المطالبة عند السداد الكامل أو الإغلاق.

سياسات وإعدادات

  • تحديد عملة لكل عملية (دين/دفعة).
  • منع الإرسال المكرر لنفس الدين في نفس اليوم.
  • إيقاف يدوي للمطالبات على مستوى الحساب أو الدين.
⚙️
اضبط تكرار التذكير، نافذة التوقيت اليومية، وقنوات الإرسال (واتساب/بريد).

نماذج الرسائل

يتغير النص تلقائيًا حسب حالة المشترك، العملة، وعدد أيام التأخير:

تأكيد دفع

مرحبًا {الاسم} 👋 تم تسجيل دفعتك {المبلغ} {العملة} بتاريخ {التاريخ}. الرصيد المتبقي: {الرصيد} {العملة}. شكرًا لك.

تنبيه متأخر

عميلنا {الاسم}، يوجد عليك {المبلغ} {العملة} متأخر منذ {أيام_التأخير} يوم. آخر مهلة: {تاريخ_المهلة}. للدفع: {رابط/واتساب}.

✍️
الحقول الديناميكية: {الاسم}، {المبلغ}، {العملة}، {التاريخ}، {الرصيد}، {أيام_التأخير}، {تاريخ_المهلة}… إلخ.

نعم—يمكن لكل عملية (دين/دفعة) اختيار عملتها، وتُعرض التقارير منفصلة لكل عملة.

هل يمكن إيقاف المطالبة عن مشترك معين؟

يوجد إيقاف يدوي على مستوى الحساب أو الدين، ويُستثنى تلقائيًا بعد السداد الكامل.

هل تُرسل رسالة تأكيد بعد الدفع؟

نعم، يُرسل إشعار واتساب/بريد يتضمن تفاصيل الدفعة ورصيد الدين بعد التحديث.
