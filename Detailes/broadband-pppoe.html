إدارة البرودباند وPPPoE

تحكّم مؤتمت بالمشتركين مع تنفيذ مباشر على السيرفر: تعطيل الحساب أو تخفيض السرعة تلقائيًا عند انتهاء الباقة، تنبيهات قبل الانتهاء، ومطالبات ديون ذكية ترتبط بحالة السداد.

كيف تعمل؟ البروفايلات والسرعات
Auto Control

تعطيل/تخفيض تلقائي

عند انتهاء الباقة يُعطَّل الحساب أو يتحول إلى بروفايل سرعة منخفضة لمنع الاستهلاك غير المشروع.

Smart Alerts

تنبيهات قبل الانتهاء

إشعارات ودّية للمشترك، وتنبيهات للمدير عبر واتساب/SMS/البريد مع جدولة وتكرار مرن.

Dunning

مطالبات ديون

رسائل بالمبلغ المستحق وعدد أيام التأخير وتاريخ الاستحقاق—وتتوقف تلقائيًا عند السداد.

كيف يعمل التحكم المؤتمت؟

  1. مراقبة الاشتراكات: تتبّع تواريخ الانتهاء وحالة PPPoE لحظيًا.
  2. التصرف عند الانتهاء: تعطيل أو تخفيض السرعة حسب سياسة الباقة.
  3. الإشعارات: إرسال قبل الانتهاء وبعده مع قنوات متعددة.
  4. الربط مع الديون: تسجيل الدين وتجديد تلقائي اختياري مع تفعيل المطالبة.
  5. التقارير: لوحات تعرض الحالات والتأخيرات يوميًا/شهريًا.
🧠
إذا جُدّد الاشتراك أو سُدِّد الدين قبل وقت التنفيذ، يتخطّى النظام إجراء التخفيض/التعطيل تلقائيًا.

سياسات البروفايل

  • تحويل تلقائي إلى Low-Speed Profile حتى السداد/التجديد.
  • إرجاع فوري إلى Profile الأصلي بعد الدفع.
  • استثناءات قابلة للتهيئة لحسابات VIP أو الساعات البيضاء.
  • منع ازدواج الجلسات والتحقق من MAC (اختياري).
⚙️
اضبط السرعات، حدّ الجلسات، ونافذة السماح من إعدادات البروفايل.
