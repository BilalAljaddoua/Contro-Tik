نظام مرن قابل للتخصيص

ControTik منصة تتكيّف مع طريقة عملك: ميزات حسب الطلب، واجهات وهوية بصرية، تنبيهات ذكية، تقارير خاصة، وتكاملات مع MikroTik وRADIUS.

خيارات التخصيص الأسئلة الشائعة
Features

ميزات حسب الطلب

بناء وحدات جديدة أو توسيع القائم: ولاء، بوابة زبائن، تقارير خاصة، POS، أو تكاملات خارجية.

  • منطق عمل مخصص للحزم/الأرباح.
  • فلترة ورسائل دين متقدمة.
  • واجهات API للاستخدام الداخلي.
UI/Branding

تخصيص الواجهات والهوية

ألوان، خطوط، ترتيب البطاقات، نِسق طباعة البطاقات، قوالب PDF وExcel بعلامتك التجارية.

  • Themes فاتح/داكن مع سيان.
  • قوالب هوتسبوت قابلة للطباعة.
  • ترويسات وفوتر الفواتير.
Automation

تنبيهات وتدفقات عمل

قواعد مخصّصة للرسائل: قبل الانتهاء، مطالبة دين، استهلاك مرتفع، أو تنبيهات صيانة للأبراج.

  • واتساب/بريد/SMS حسب الحدث.
  • محتوى ديناميكي للرسائل.
  • جداول تكرار مرنة.
MikroTik / RADIUS

تكاملات الشبكات

تنفيذ أوامر على الراوتر، بروفايلات ديناميكية، PPPoE/Hotspot، وربط مع RADIUS/Radius Manager.

  • تعطيل تلقائي/تحويل لبروفايل ضعيف.
  • إنشاء بطاقات هوتسبوت مع باركود.
  • سجل عمليات مرتبط بالمشترك.
Reports

تقارير وExports مخصّصة

جداول مخصّصة، أعمدة حسب الطلب، تنسيقات PDF/Excel محمية، وتجميع حسب البرج/العملة/الفترة.

  • حسابات شراكة مرنة.
  • تقارير ديون ومطالبة ذكية.
  • لوحات KPIs مناسبة لفريقك.

آلية تنفيذ التخصيص

  1. تحليل المتطلبات: نجمع حالات الاستخدام، الصلاحيات، والتقارير المطلوبة.
  2. النطاق والجدول: نحدّد ما سيتم تنفيذه ومراحله ونقاط التسليم.
  3. نموذج أولي: معاينة سريعة للواجهات/التقارير لأخذ الملاحظات.
  4. التطوير والاختبار: بناء الميزة وفق معايير الأداء والأمان.
  5. الإطلاق والمتابعة: تدريب مختصر ودعم بعد الإطلاق لتحسين التجربة.
🧪
يمكن توفير بيئة تجريبية Staging قبل الإطلاق لضمان سلاسة الدمج مع بياناتك الحقيقية.

خدمة التخصيص المدفوعة

لمتطلبات معقّدة أو تجربة مخصّصة بالكامل، نوفر نسخة تفصيلًا على قياس عملك.

  • سحابة خاصة/استضافة داخلية.
  • اتفاقية مستوى خدمة (SLA).
  • دعم أولوية وتدريب الفريق.
ابدأ محادثة سريعة لنعرف بالضبط ماذا تريد تخصيصه.

هل أحتاج سيرفر خاص؟

ليس بالضرورة. يمكن الاستضافة السحابية أو بيئتك الداخلية حسب سياسة شركتك.

يدعم تكاملات عبر API/Exports، ويمكن تنفيذ موصلات مخصصة لأنظمتك الحالية.
