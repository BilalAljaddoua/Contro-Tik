نظام إدارة ممتلكات الشبكة

جرد شامل للأصول ( راوتر، كاميرا، سويتش...) مع سجلات حركة، مواقع وحالات استخدام، مستخدم حالي ورقم الاتصال، وتنبيهات صيانة—وتقارير Excel محمية.

المزايا الأساسية أدوات الإدارة
Inventory

توثيق شامل لكل قطعة

النوع، تاريخ الشراء والقيمة، الموقع (برج/مستودع)، حالة الاستخدام (فعّال/معطل/احتياطي)، المستخدم الحالي والهاتف، مدة الكفالة والملاحظات.

Movements

سجل حركة ذكي

نقل مواقع، تغيير حالات، تعديل قيم، تحديث المستخدم/الهاتف، وتوثيق الأعطال والإصلاحات بالطابع الزمني وسجل تدقيق.

Capital

تحليل رأس المال

قيمة الأصول حسب الموقع أو الفئة، إجمالي الفاقد/المعطل، وتصدير إلى Excel محمي للاعتماد الرسمي.

أدوات إدارة وفلترة

  • فلاتر حسب النوع، الحالة، الموقع، المستخدم، أو الفئة (راوتر، كاميرا…).
  • وسوم ذكية لتجميع الأجهزة (مثل: «حرج»، «يتطلب صيانة»).
  • طباعة قوائم الجرد أو تصديرها إلى Excel مع حفظ شروط الفلترة.

تنبيهات صيانة

إشعار تلقائي عند اقتراب انتهاء الكفالة أو تسجيل عطل جديد، مع إسناد المهمة لمدير البرج أو الفني المسؤول.

جرد محكم، سجلات حركة، وتنبيهات صيانة بلا تعقيد.
