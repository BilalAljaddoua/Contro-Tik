إضافة الموظفين وإدارة الرواتب

أضِف عددًا غير محدود من الموظفين مع الاسم ورقم الهاتف والراتب الشهري. يُنشئ النظام راتبًا شهريًا تلقائيًا لكل موظف، ويدعم سحب الراتب أو سلفة على الراتب القادم أو ترحيل جزء للراتب التالي—مع أدوار وصلاحيات وتعيين فنيين على الأبراج.

المزايا الأساسية خطوات الإضافة
Unlimited

إضافة غير محدودة

إنشاء موظفين بعدد كبير دون قيود—حفظ الاسم ورقم الهاتف والراتب الشهري.

Auto Payroll

راتب شهري تلقائي

ينشئ النظام قيد راتب لكل شهر تلقائيًا مع إمكانية التعديل قبل الاعتماد.

Cashflow

سحب وسُلَف وترحيل

سحب راتب، سلفة على الراتب القادم، أو ترحيل جزء إلى الشهر التالي—بضغطة واحدة.

Roles

أدوار وتعيين فنيين

تخصيص دور لكل موظف (مدير/موظف/محاسب/فني) وربطه كفني ببرج معيّن لمهام الصيانة والتركيب.

Ledger

سجل سحوبات مفصل

كل عملية سحب/سلفة تُسجل بالتاريخ والمبلغ والملاحظات مع إشعارات واتساب فورية.

Control

تعديل/حذف/تراجع

إمكانية التعديل والحذف والتراجع عن العمليات مع سجل تدقيق واضح.

خطوات إضافة موظف

  1. الإدارة ⇢ الموظفون ⇢ إضافة موظف.
  2. أدخل الاسم ورقم الهاتف والراتب الشهري.
  3. اختر الدور (مثلاً: فني، محاسب…) ويمكن تعيينه على برج معيّن.
  4. حفظ—يُنشئ النظام راتب الشهر تلقائيًا مع ربطه بالموظف.
🧠
يدعم تعدد العملات للرواتب إن لزم، وتُضمَّن الفلاتر والفترة عند التصدير.

عمليات الراتب المدعومة

  • سحب راتب كامل/جزئي مع رصيد متبقٍ.
  • سلفة تُخصم تلقائيًا من الراتب القادم.
  • ترحيل جزء من الراتب للشهر التالي.
  • اعتماد نهائي وإشعار واتساب للموظف.
📲
تصل إشعارات فورية عبر واتساب عند السحب/السلفة/اعتماد الراتب مع تفاصيل العملية.

هل يمكن إضافة عدد كبير من الموظفين؟

نعم، النظام يدعم إضافة عدد كبير دون قيود عملية.

هل ينشئ الراتب تلقائيًا كل شهر؟

يُنشئ قيد الراتب الشهري تلقائيًا لكل موظف مع إمكانية المراجعة قبل الاعتماد.

هل توجد سلف وترحيل للراتب؟

يمكن عمل سلفة على الراتب القادم أو ترحيل جزء من الراتب للشهر التالي بشكل مباشر.

هل يوجد سجل تفصيلي وإشعارات؟

كل العمليات تُسجل بالتفصيل وترافقها إشعارات واتساب فورية، مع إمكانيات تعديل/حذف/تراجع.
